# QuickMart

Hyperlocal multi-vendor marketplace with ultrafast delivery.

## Concept

A Blinkit + Swiggy Instamart hybrid. Orders are fulfilled by nearby vendors:
- **⚡ 30-min delivery** if a vendor is close
- **🕒 Estimated time** otherwise
- **No central warehouse** — true multi-vendor system

## Tech Stack

- React + Vite
- Tailwind CSS

## Getting Started

```bash
npm install
npm run dev
```

## Features

- Location-based vendor discovery
- Dynamic delivery time based on distance
- Multi-vendor product catalog
- Real-time cart management
